# 프로젝트 개요

## 서비스 개요

**Date Diary**는 데이트 사진들을 업로드하면 AI가 자동으로 감성적인 일기를 생성해주는 서비스입니다.

## 핵심 가치 제안

- **간편한 기록**: 여러 장의 사진만으로 데이트 추억을 쉽게 기록
- **AI 감성 일기**: 사진 분석을 통해 자동으로 감성적인 일기 생성
- **추억 보관**: 소중한 데이트 순간들을 체계적으로 저장

## 타겟 사용자

- 20~30대 연인들
- 데이트 추억을 기록하고 싶지만 일기 쓰기가 번거로운 사용자
- SNS보다 프라이빗하게 추억을 보관하고 싶은 커플

## 핵심 사용 시나리오

1. 사용자가 데이트에서 찍은 사진 2~10장을 업로드
2. AI가 사진을 분석하여 장소, 활동, 분위기 파악
3. AI가 감성적인 일기 형식의 텍스트 자동 생성
4. 사용자가 생성된 일기를 확인하고 필요시 수정
5. 일기 저장 및 나중에 달력 형태로 조회

## MVP 범위

### 포함

- 사진 업로드 (2~10장)
- AI 일기 자동 생성
- 일기 목록 조회
- 일기 상세 조회
- 간단한 인증 (이메일/비밀번호)

### 제외 (향후 확장)

- 소셜 로그인
- 일기 공유 기능
- 커플 계정 연동
- 일기 수정 기능
- 캘린더 뷰
- 검색 기능
- 사진 편집 기능
- 댓글/이모티콘 기능

## 성공 지표

- 사용자 등록 수
- 일기 생성 완료율
- 일주일 내 재방문율
- 평균 업로드 사진 수
