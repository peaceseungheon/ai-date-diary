# 유저 스토리

## 사용자 역할

- **일반 사용자**: 서비스를 사용하는 개인
- **커플 사용자**: 파트너와 계정을 연동한 사용자 (Post-MVP)

---

## Epic 1: 회원 관리

### US-001: 회원가입

**As a** 신규 사용자  
**I want to** 이메일과 비밀번호로 회원가입하고  
**So that** 내 데이트 일기를 기록할 수 있다

**인수 조건**:

- [ ] 이메일, 비밀번호, 닉네임을 입력할 수 있다
- [ ] 이메일 형식이 검증된다
- [ ] 비밀번호는 최소 8자 이상이어야 한다
- [ ] 중복된 이메일은 사용할 수 없다
- [ ] 가입 완료 후 자동으로 로그인된다

**우선순위**: 높음  
**스토리 포인트**: 3

---

### US-002: 로그인

**As a** 기존 사용자  
**I want to** 이메일과 비밀번호로 로그인하고  
**So that** 내 일기를 확인할 수 있다

**인수 조건**:

- [ ] 이메일과 비밀번호를 입력할 수 있다
- [ ] 잘못된 정보 입력 시 명확한 오류 메시지를 본다
- [ ] 로그인 성공 시 일기 목록 페이지로 이동한다
- [ ] 로그인 상태가 유지된다 (JWT 토큰)

**우선순위**: 높음  
**스토리 포인트**: 2

---

### US-003: 로그아웃

**As a** 로그인된 사용자  
**I want to** 로그아웃하고  
**So that** 내 계정을 안전하게 보호할 수 있다

**인수 조건**:

- [ ] 로그아웃 버튼을 클릭할 수 있다
- [ ] 로그아웃 시 로그인 페이지로 이동한다
- [ ] 로그아웃 후 인증이 필요한 페이지에 접근할 수 없다

**우선순위**: 중간  
**스토리 포인트**: 1

---

## Epic 2: 일기 생성

### US-004: 사진 업로드

**As a** 사용자  
**I want to** 데이트에서 찍은 사진 2~10장을 업로드하고  
**So that** AI가 일기를 생성할 수 있다

**인수 조건**:

- [ ] 파일 선택 버튼을 클릭하여 사진을 선택할 수 있다
- [ ] 한 번에 여러 장을 선택할 수 있다 (최소 2장, 최대 10장)
- [ ] 선택한 사진들의 미리보기를 볼 수 있다
- [ ] 업로드 중 진행률을 확인할 수 있다
- [ ] 개별 사진을 삭제하고 다시 추가할 수 있다
- [ ] JPG, PNG, HEIC 형식만 업로드할 수 있다
- [ ] 파일 크기 제한(10MB/장)을 초과하면 경고 메시지를 본다

**우선순위**: 높음  
**스토리 포인트**: 5

---

### US-005: AI 일기 생성 및 확인

**As a** 사용자  
**I want to** 업로드한 사진들을 분석하여 AI가 감성적인 일기를 생성해주고  
**So that** 쉽게 추억을 기록할 수 있다

**인수 조건**:

- [ ] "일기 생성" 버튼을 클릭할 수 있다
- [ ] 데이트 날짜를 선택할 수 있다 (기본값: 오늘)
- [ ] AI 생성 중 로딩 상태를 확인할 수 있다
- [ ] 생성된 일기 제목과 본문을 미리 볼 수 있다
- [ ] 생성 시간은 30초 이내여야 한다
- [ ] 생성 실패 시 재시도할 수 있다

**우선순위**: 높음  
**스토리 포인트**: 8

---

### US-006: 일기 저장

**As a** 사용자  
**I want to** AI가 생성한 일기를 저장하고  
**So that** 나중에 다시 볼 수 있다

**인수 조건**:

- [ ] "저장" 버튼을 클릭할 수 있다
- [ ] 저장 성공 시 일기 상세 페이지로 이동한다
- [ ] 저장 실패 시 오류 메시지를 확인할 수 있다
- [ ] 저장 중 다른 페이지로 이동하면 경고 메시지를 본다

**우선순위**: 높음  
**스토리 포인트**: 3

---

## Epic 3: 일기 조회

### US-007: 일기 목록 보기

**As a** 사용자  
**I want to** 내가 작성한 모든 일기 목록을 보고  
**So that** 과거의 데이트를 회상할 수 있다

**인수 조건**:

- [ ] 일기들이 데이트 날짜 최신순으로 정렬되어 표시된다
- [ ] 각 일기의 썸네일, 제목, 날짜, 본문 미리보기를 볼 수 있다
- [ ] 일기를 클릭하면 상세 페이지로 이동한다
- [ ] 20개씩 페이지네이션 또는 무한 스크롤로 로딩된다
- [ ] 일기가 없을 때 안내 메시지를 본다

**우선순위**: 높음  
**스토리 포인트**: 5

---

### US-008: 일기 상세 보기

**As a** 사용자  
**I want to** 특정 일기의 전체 내용과 모든 사진을 보고  
**So that** 그 날의 추억을 자세히 회상할 수 있다

**인수 조건**:

- [ ] 일기 제목, 전체 본문, 데이트 날짜를 볼 수 있다
- [ ] 모든 사진을 갤러리 형태로 볼 수 있다
- [ ] 사진을 클릭하면 전체 화면으로 확대할 수 있다
- [ ] 사진 간 좌우 스와이프로 이동할 수 있다
- [ ] 뒤로 가기 버튼으로 목록으로 돌아갈 수 있다

**우선순위**: 높음  
**스토리 포인트**: 5

## Epic 4: 일기 관리 (Phase 2)

### US-009: 일기 수정

**As a** 사용자  
**I want to** 저장한 일기의 내용을 수정하고  
**So that** 더 정확하거나 개인적인 표현으로 바꿀 수 있다

- [ ] 제목과 본문을 편집할 수 있다
- [ ] 날짜를 변경할 수 있다
- [ ] "AI 재생성" 옵션을 선택할 수 있다
- [ ] 수정 사항을 저장하거나 취소할 수 있다

**우선순위**: 중간

## Epic 5: 캘린더 및 통계 (Phase 2)

### US-011: 캘린더 뷰

**As a** 사용자  
**I want to** 달력 형태로 일기를 보고  
**So that** 특정 날짜의 데이트를 쉽게 찾을 수 있다

**인수 조건**:

**스토리 포인트**: 5

**As a** 사용자  
**I want to** 나의 데이트 통계를 보고  
**So that** 우리의 관계를 돌아볼 수 있다

**인수 조건**:

- [ ] 월별 데이트 횟수 그래프를 볼 수 있다
- [ ] 가장 많이 방문한 장소 유형을 볼 수 있다
- [ ] 첫 데이트 이후 경과 일수를 볼 수 있다

**우선순위**: 낮음  
**스토리 포인트**: 5

## Epic 6: 커플 기능 (Phase 3)

### US-013: 커플 연동

**As a** 사용자  
**I want to** 파트너와 계정을 연동하고  
**So that** 서로의 일기를 공유할 수 있다

**인수 조건**:

- [ ] 초대 코드를 생성할 수 있다

**I want to** 파트너가 작성한 일기를 보고

- [ ] 파트너 일기는 수정/삭제할 수 없다

**I want to** 일기에 태그를 추가하고 필터링하고

## 우선순위 정의

- **높음 (High)**: MVP에 필수적인 기능
- **중간 (Medium)**: Phase 2-3에서 추가
- **낮음 (Low)**: Phase 4-5에서 추가

## 스토리 포인트 가이드

- 1-2: 간단 (1-2일)
- 3-5: 보통 (3-5일)
- 8: 복잡 (1주)
- 13+: 에픽 분할 필요
