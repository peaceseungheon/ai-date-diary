# 개발 로드맵

## Phase 1: MVP 개발 (4-6주)

### Week 1-2: 프로젝트 설정 및 기본 인프라

- [ ] 프로젝트 초기 설정
  - Git 레포지토리 생성
  - Frontend 프로젝트 생성 (Vite + React + TypeScript)
  - Backend 프로젝트 생성 (Express + TypeScript)
  - 데이터베이스 스키마 설정 (Prisma)
- [ ] 개발 환경 구축

  - 환경변수 관리
- [ ] 기본 인프라 설정
  - AWS S3 버킷 생성

## 성공 지표 (KPI)

### MVP Phase

- [ ] 첫 100명 사용자 달성
- [ ] 일기 생성 완료율 > 80%
- [ ] 주간 활성 사용자(WAU) > 30%

### 성장 Phase

- [ ] 월간 활성 사용자(MAU) 1,000명
- [ ] 평균 사용자당 월 4개 일기 작성
- [ ] 커플 연동률 > 50%

### 장기 목표

- [ ] MAU 10,000명
- [ ] 누적 일기 수 50,000개
- [ ] 유료 기능 전환율 > 5%
